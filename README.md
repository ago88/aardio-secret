# aardio-secret
一款aardio开发的自用帐户密码生成密文及解密工具

使用了多个可逆的加解密算法，如AES\AUTHCODE\PASSWORD等，及OTP一次性密码，PC端及服务端放置不同算法的密钥
> 服务端采用了C大的RPC服务 [RPC服务端](https://github.com/carlaau/aardio-erp/tree/master/aardio-rpc)
